# Model Assets

This directory is populated by running:

```bash
python scripts/download_models.py
```

The script downloads pre-trained OpenVINO IR files from the Open Model Zoo. These models are licensed under **Apache-2.0** and are suitable for commercial use.

Models used:
- `person-detection-0200` (FP16-INT8)
- `human-pose-estimation-0007` (FP16)
- `hand-pose-estimation-0001` (FP16)

See `docs/LICENSING.md` for the authoritative license links.
